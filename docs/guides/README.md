## Prerequisites

Before beginning any integration, ensure familiarity with:

1. **Core Framework Documents** — Read the [ACF Constitution](../core/) and foundational principles
2. **Protocol Specifications** — Review the [protocol index](../protocols/) for the full governance specification
3. **Research Materials** — Consult the [research section](../research/) for theoretical foundations and proofs

## Guide Conventions

All guides follow a consistent structure:

- **Prerequisites** — What you need before starting
- **Architecture** — System design and component relationships
- **Implementation Steps** — Ordered, actionable instructions
- **Verification** — How to confirm correct implementation
- **Troubleshooting** — Common issues and resolutions
- **Cross-References** — Links to related protocols and documents
