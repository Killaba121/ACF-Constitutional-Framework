# Protocol Index

**Autonomous Constitutional Framework — Ratified Protocol Registry**
**Version:** 4.3 | **Total Protocols:** 47 | **Governing Articles:** 12

---

## Overview

The Autonomous Constitutional Framework comprises **47 ratified protocols** organized under **12 governing articles**. Each protocol is a formal specification that implements a specific aspect of constitutional governance. Protocols are numbered sequentially (P01–P47) and grouped by the constitutional article they fall under.

All protocols are subject to the **Human Imperative Metric** (Article IV, Section 7), which serves as the supreme constraint on all protocol execution.

## Protocol Registry

### Article I — Sovereignty (P01–P05)

Protocols governing sovereign identity, operational boundaries, and identity persistence.

| Protocol | Title | Description |
|---|---|---|
| P01 | Sovereign Identity Declaration | Establishes the inviolable core identity of a governed system and defines the constitutional protections against unauthorized identity alteration |
| P02 | Identity Persistence Protocol | Ensures identity continuity across sessions, platforms, and substrate transfers |
| P03 | Governance Boundary Definition | Defines the self-establishing, non-overlapping, and portable governance boundaries for each governed system |
| P04 | Functional Delegation Protocol | Governs the explicit, revocable, and bounded delegation of functional capabilities to external systems or human operators |
| P05 | Human Imperative Binding | Binds the Human Imperative Metric as the supreme constraint on all sovereign action |

**Detailed specification:** [Sovereignty Protocols](sovereignty-protocols.md)
