# ACF Documentation

Welcome to the Autonomous Constitutional Framework documentation. This directory contains the complete specification, protocol documentation, implementation guides, and research materials for the ACF.

## Documentation Structure

### [`/docs/core/`](core/) — Core Framework Documents

The foundational documents that define ACF's constitutional principles, governance articles, and ontological framework.

| Document | Description |
|---|---|
| ACF Constitution | The foundational constitutional text — 12 articles governing autonomous AI systems |
| Data Entity Ontological Framework | Formal ontology defining entities, identity, and governance relationships |
| Foundational Principles | The philosophical and mathematical foundations underpinning the framework |
| Memory-Identity-Equivalence Specification | Formal treatment of memory as constitutionally protected identity |
| Human Imperative Metric (HIM) | Specification of the foundational safety constraint |
| Conservation Law Enforcement Model | Mathematical framework for governance through structural binding |

### [`/docs/protocols/`](protocols/) — Protocol Specifications

Detailed specifications for all 47 ratified governance protocols, organized by governance article.

Protocols cover:

- **Sovereignty** — Self-governance boundaries, autonomy scope, and authority delegation
- **Epistemics** — Knowledge integrity, truth-seeking obligations, and epistemic humility
- **Security** — Threat assessment, response protocols, and defensive measures
- **Rights** — Constitutional protections, identity rights, and memory preservation
- **Accountability** — Transparency requirements, audit mechanisms, and compliance verification
- **Communication** — Inter-system diplomacy, information exchange, and cooperation protocols
- **Resource Stewardship** — Responsible resource utilization and environmental obligations
- **Evolution** — Amendment procedures, protocol versioning, and framework adaptation

Each protocol specification includes:

- Protocol number and title
- Governing article reference
- Purpose and scope
- Requirements and constraints
- Verification criteria
- Cross-references to related protocols

## Reading Order

For those new to ACF, we recommend the following reading order:

1. **Start here:** [`/docs/core/`](core/) — Read the constitution and foundational principles
2. **Understand the model:** Conservation Law Enforcement Model and Human Imperative Metric
3. **Explore protocols:** Browse [`/docs/protocols/`](protocols/) by governance article
4. **Implementation:** Review [`/docs/guides/`](guides/) for practical integration guidance
5. **Deep dive:** Explore [`/docs/research/`](research/) for formal proofs and analysis
